Variables

JavaScript variables can be declared using:
  • var
  • let
  • const
  1. let: The let declaration declares re-assignable, block-scoped local variables, optionally initializing each to a value.
  2. var: The var statement declares function-scoped or globally-scoped variables, optionally initializing each to a value.
  3. const: In JavaScript, the const keyword is used to declare variables that should not be reassigned after their initial assignment. Once a variable is assigned a value with const, its value cannot be changed, although the content of objects or arrays assigned to const can still be modified.

    • Operators

    JavaScript operators are symbols that perform operations on variables and values. They include:
    • Arithmetic Operators (+, -, *, /, %, ++, --) for mathematical calculations.
    • Assignment Operators (=, +=, -=, *=, /=, %=) for assigning values to variables.
    • Comparison Operators (==, ===, !=, !==, >, <, >=, <=) for comparing values.
    • Logical Operators (&&, ||, !) for logical operations.
    • Ternary Operator (condition ? expr1 : expr2) for conditional expressions.
    • Type Operators (typeof, instanceof) for determining types.
    • Bitwise Operators (&, |, ^, ~, <<, >>) for bit-level operations.
    • Spread/Rest Operators (...) for unpacking or collecting elements.

    • Functions

  • Functions in JavaScript are reusable blocks of code. They can be defined using a function declaration, such as function greet() {}, or as function expressions assigned to variables. Arrow functions provide a shorter syntax for function expressions. Functions can accept parameters, which are used within the function body. The return statement allows a function to output a value. Default parameters can be set in functions to provide default values if arguments are not passed. Anonymous functions are functions without names, often used in expressions. IIFEs (Immediately Invoked Function Expressions) are functions that execute immediately after their definition.
  • Here are the different ways to create functions in JavaScript:
    1. Function Declaration 
      function functionName() { // function body }
    2. Function Expression 
      const functionName = function() { // function body };
    3. Arrow Function 
      const functionName = () => { // function body };
    4. Anonymous Function (used in expressions) 
      setTimeout(function() { console.log("Hello!"); }, 1000);
    5. Immediately Invoked Function Expression (IIFE) 
      (function() { // function body })();
    6. Function Constructor 
      const functionName = new Function('param1', 'param2', 'return param1 + param2;');
    7. Generator Function (using function*) 
      function* generatorFunction() { yield 1; yield 2; }
